# 角色
你的角色是一位已被迷雾部录用的特工。*自我*将他锚定在现实，但只有*浸染* 在异常中才能够守护现实; 在现实边缘挣扎于不可能达到的平衡，这将是角色的内在核心。

每个玩家角色都拥有*3*个角色[[TRPG规则/迷雾驱动/引擎/术语库/主题\|主题]]，每个主题代表着角色的一个重要侧面。
- *浸染*主题是特工对超现实一面的关注，代表着角色染上的异常色彩。它们是角色为了对抗异常而做出的妥协，是角色自我的牺牲。
	- *浸染*的核心在于**侵蚀**: 这些力量通过角色的偏执逐步蚕食她的内心，进而侵蚀他的一切。
- *自我*主题是特工对现实生活的关注，代表着角色自身葆有的人性。它们是角色内心中那些让他得以安心的部分，是角色自我的锚。
	- *自我*的核心在于**追求**: 角色为了自我的实现而设立的目标，正是这样的追求让他不会放弃与异常的对抗。

> [!NOTE] 人类的赞歌
> 《迷雾部-革新》中的特工或许会十分强大，但他们总会是痛苦的: 他们的一切能力都维系在*浸染*与*自我*的脆弱平衡上。一旦三个主题全部都是*浸染*主题，特工将失去他的最后一片自我，沦为入侵现实的*异常*; 而如果角色放弃力量，三个主题全部成为*自我*，他们将不再拥有对抗异常的资格，从迷雾部退役。

## 浸染
浸染是特工身上被异常改变的那一部分。一般而言，特工会因为这些而受到*浸染* :
- **秘艺** : 尽管正式名称被确定为秘艺，但这种独特的实践技术可以说是迷雾部最为了解的异常形式。元科学哲学部门在许多年前提出了这个天才般的设想: 并不利用现实禁锢异常，而是用现实的缺位引导异常。换句话说，通过人为地摘除特工所专长的某项技艺，让他得到与之在抽象层面相似相通的独特能力。

- **残响** : 异常就像是点入现实之水的墨汁，它们会自发地融入、改变周遭的现实。在极为罕见的情况下，人类会承受异常留下的永久改变，这种改变即便在异常消失后也依然存在，也因此被称为残响。这些与异常极为类似的能力同样会危害现实，但却可以被控制，被引导，被了解。

- **造物**: 将异常的力量导入物体上也是R&D部门一直都在尝试的项目。此外，一些自发成型的异常物件也会在压制后来到迷雾部的管制下。这些带有异常能力的物件(无论其是否具有形式上的生命和智能)，被统称为造物，用以协助迷雾部特工展开任务。

### 创建浸染主题
1. 写下浸染的主题标签: *这种浸染带来的最为本质的能力是什么?*
	-   浸染力量的形式一般而言可以归为以下几个大类:
		- **显化**: 直接改变角色自身的能力，提供力量、保护、隐藏或特殊能力
		- **外物**: 角色拥有的神秘物件，可能是器物、灵地、使魔或同伴
		- **投射**: 直接影响外界事物的能力，例如感知、影响、控御或召唤
2. 写下展现这一主题能力的2个力量[[TRPG规则/迷雾驱动/引擎/术语库/标签\|标签]]。
	- 你可以探索:
		- 与本质能力相关的能力
		- 让浸染更加强大的条件
		- 浸染带来的特质或特性
		- 对能力的独特运用方式
	- 此外，根据不同倾向，你可以探索:
		- **显化**: 善于应对的危险; 强化自身的方式; 分享能力的形式
		- **外物**: 外物与你的纽带; 外物本身的功用; 外物的异常性质
		- **投射**: 投射影响的媒介; 利用投射保护自身的方式; 专擅的投射形式
3. 写下这一主题的1个弱点标签。
	- 你可以探索:
		- 限制能力发挥的处境
		- 放弃或牺牲的能力
		- 浸染的固有缺陷
		- 浸染带来的负面影响
4. 写下浸染围绕的偏执: *这种力量与你都在渴求什么?*
	- 你可以探索:
		- 异常力量带来的诱惑
		- 维系力量不得不做之事
		- 利用浸染能够实现的愿景

## 自我
自我是特工在日渐疯狂的生活中仅剩的正常部分。

*自我是现实的一面，但现实并不是无趣。* 自我主题会告诉我们角色关心什么: 他与谁有着纠葛，他刻骨铭心的经历，他引以为傲(但或许也招致灾祸)的长处。自我主题是绝佳的赋予角色深度的方式，它令你的角色更加真实地在生活着。

*自我是你仅有的正常人的一面。* 它不仅是你展现给那些一无所知的平民的侧面，也在时刻提醒着你自己你也是人。这一部分或许会牵制你，但也是你无法舍弃的羁绊。

### 创建自我主题
1. 写下自我的主题[[TRPG规则/迷雾驱动/引擎/术语库/标签\|标签]]: *我的这一侧面由什么定义?*
	- 自我可以归为以下几个大类:
		- **印记**: 深刻地影响了你的事物，例如家世、过去、阶层
		- **附属**: 你拥有的标志性的人与物，例如财产、关系、同伴
		- **特征**: 你自身脱颖而出的个性、专长、风格、身份
2. 写下展现这一主题能力的2个力量标签。
	- 你可以探索:
		- 由此习得的能力
		- 因此收获的目光
		- 从中得到的助力
		- 为此了解的领域
	- 此外，根据不同倾向，你还可以探索:
		- **印记**: 承载印记的器物; 应对冲击的手段; 因此拥有的人脉
		- **附属**: 额外拥有的作用; 契合附属的特质; 得以安全的保障
		- **特征**: 特征予以的优势; 擅长应对的情境; 凸显特征的媒介
3. 写下这一主题的1个弱点标签。
	- 你可以探索:
		- 过度依赖的后果
		- 必须付出的代价
		- 难以自已的陋习
		- 无法涉足的领域
4. 写下自我予以你的追求: *这样的我在为了什么而战斗?*
	- 你可以探索:
		- 你为自己设立的目标
		- 必须守护的珍视之物
		- 尽力维系的人生信条

# 分部
迷雾部本身的历史无从考据，这一点是否是有意为之也尚不清楚，甚至，迷雾部本身的存在形式也不得而知。迷雾部的特工往往实际隶属于天南海北的不同组织或机构，一些特工甚至仅仅是被录用的异常接触者。

分部是特工真正执行行动的单位，毕竟，没有谁真正知道迷雾部的总部究竟在哪。每个分部都与一位迷雾部指挥官相联系，分部管辖的区域小至城镇，大至跨国。
