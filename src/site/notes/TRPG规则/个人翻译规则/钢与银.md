# 创建你的猎魔人
1. 炼金突变将你转变为完美的狩魔猎人，却也留下了非人的*印记*。写下你自己的印记，或从下面选择若干个印记:
	- 猫眼
	- 干尸的皮肤
	- 精灵耳朵
	- 鳞片
2. 你的训练与古老的魔法仪式令你精通于两项*技艺*，写下你的技艺，或从下面选择: 战斗，谋生，调查，炼金， 潜行，运动，操弄，骑术，设陷，猎魔感官，艺术
3. 选择2-5中的一个*能力数字*。
	- 数字越高你越擅长银的领域, 亦即所有与非自然生物有关的事物.
	- 数字越低你越擅长钢的领域，亦即所有与人类有关的事物。
	- 这个数字代表了你对战术、追踪、交涉、操弄、知识、弱点和个性的了解。
4. 你拥有3点*装备&补给*，在每次安全的休息过后都会完全补充。这可以代表任何可能的装备，例如: 药剂与炼金炸弹，毒药， 攀爬用具，狩猎的战利品，甚至用于打点关系的钱财。
5. 你拥有3点*心智&意志*，代表着抵御肉体、心智以及社交场合的风险的能力。这些点数只有特殊情况下才会使用，具体场景取决于GM。
6. 为你的猎魔人取个名字，描述一下钢剑与银剑的模样。

> [!tip]
> 每次完成委托后，你都会获得钱币。
> 钱币可以用于学习新的技艺，或是增加装备&补给的最大点数。

# 承担风险
作为猎魔人，你熟习战斗、法印与各种怪物的狩猎技巧。 当你的行动需要承担风险时:
- 掷1D6来看看发生了什么
- 如果你有适合的*技艺*，额外掷1D6
- 如果你有所**准备**，或是描述你如何使用*装备*并花费对应点数，额外掷1D6

将每个骰子的结果与你的*能力数字*比较:
- 如果这是为钢掷骰(人类)，高于数字的骰值视为成功。
- 如果这是为银掷骰(怪物)，低于数字的骰值视为成功。

如果:
- **没有骰子成功**，你的行动就失败了。失去1点心智&意志，或是花费1点装备&补给来规避。GM将描述情况如何急转直下。
- **只有1个成功**，你达到了部分成功。你如愿地达成了目标，但GM会描述情况如何变得更加复杂。
- **有2个成功**，你就如愿达成了目标!
- **有3个成功**，你达成了重大成功! 你如愿以偿，并且主持人会向你展现一个绝佳的**机会**。

> [!tip]
> 如果骰值恰好等于你的能力数字，你就获得了猎魔时刻! 你得以重整旗鼓，斗志重燃。这个骰子将算作1个成功，并且你能够恢复1点心智&意志。

# 主持游戏
作为主持人, 你的职责是描绘一个充满了战争、腐败、怪物，以及艰难的道德抉择的中世纪奇幻世界。

将猎魔人想象成黑色侦探，受雇调查一桩骇人听闻的罪案，找出罪犯，将其绳之以法。在这个世界上，没有人是纯粹的善或恶; 每个决定都举足轻重，其后果往往也和背后的道德一样灰色而模糊。

农民在战争和贵族的冷漠中挣扎求生，缺乏直面怪物或是为自己复仇的力量。他们最多只能凑几枚钱币，卖掉珍贵物品或背上债务，只为雇佣一个猎魔人。

委托部分的表可以作为创意工具，帮助生成关键的故事元素。如果某个表引入了新的怪物，可以再次掷骰查看其他表以获取更多细节。这造就了一个充满活力、相互关联的世界，委托层层叠加着复杂的人物和不断演变的事件。

通过游玩来发现角色如何与世界互动与解决问题。角色们都是专家，只有在真正危险时才需要掷骰子。

如果涉及超自然生物，就是银骰。否则，采用钢骰。仅此而已。

为每个角色创造以自己的方式引人瞩目的机会。和你的玩家一起构建世界。询问角色对地点或怪物的了解，并将他们的回答融入故事之中。

这种协同创作是体验的核心。

当玩家感到迷茫时，引入新的线索。如果节奏变慢，触发怪物伏击或突发的危险。

时刻谨记: 这个世界残酷而严峻，危机永远如影随形。
