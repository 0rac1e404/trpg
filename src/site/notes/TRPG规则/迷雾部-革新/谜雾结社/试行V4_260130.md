# 游戏结构
我们将直接切入角色们完成大部分调查之后、开始推理真相并做出计划的时刻。

**再演绎**允许角色再次审视自己的调查和探索，从中发现更多的线索; **检验**则允许角色对自己的推断展开验证，抓住直面异常的机会。

细节会随着**再演绎**一层层完善，时间将随着**检验**一步步推进，二者共同促使角色完成书写档案，直面异常。

# 书写档案v4
当玩家开始尝试回答以下两种问题之一时，他们的角色即是在试图*书写档案*:
- *特性问题*: 异常如何作用于现实?
- *本质问题*: 异常是什么?

 回答时，玩家需要叙述自己的结论，检查与线索相关的结论:
- 每条能证明结论的线索将给予+1效力
	- 此外，能够证明结论的线索数量至少需要与谜团的复杂等级相当
- 每条与结论冲突的线索将给予-1效力

然后，选择一种书写档案的形式: 
- *再演绎*: 角色已经在之前的探索中经历了能揭示真相的场景，现在将之再度演绎，**掷骰**以推知真相与可能曾引发的后果
- *检验*: 凭借已经掌握的线索，角色开始对结论的验证，**掷骰**以发现真相与可能出现的*机会*和*代价*
	- 只有*检验*才可能出现让角色完全处理异常的*机会*，因此，玩家们一定需要通过检验来完成书写档案。

## 再演绎
再演绎会导向一个过去的场景: 角色在那时收集到了相关线索，即将经历的事情足以揭示真相。与MC一起构建这个场景，**描述**角色在做什么，然后开始**掷骰**:
- 如果最终结果为...
	- 5及以下: 失败! 
		- 结论完全错误，因为发现了**一条足以反驳的线索**
		- 演绎中的举动在如今的审视下，造成了**不利于当下的后果**
	- 6-8: 混合成功。
		- 结论正确，并且可以作为进一步推理的线索
		- 演绎中的举动在如今的审视下，造成了**不利于当下的后果**
	- 9及以上: 成功!
		- 结论正确，并且可以作为进一步推理的线索
## 检验
检验是角色为了验证结论进行的实验: 凭借已经掌握的相关线索，角色现在开始行动，根据结果判断结论的正确与否。与MC一起构建这个场景，**描述**角色的计划是什么，然后开始**掷骰**:
- 如果最终结果为...
	- 5及以下: 失败! 
		- 结论完全错误，因为发生了足以反驳结论的**后果**
		- 并且，这个后果会揭示一个额外的让异常更加棘手的特性
	- 6-8: 混合成功。
		- 结论正确，并且可以凭借这个*机会*直接应对异常
		- 但是，**后果**出现，并揭示一个额外的让异常更加棘手的特性
	- 9及以上: 成功!
		- 结论正确，并且可以凭借这个*机会*直接应对异常
